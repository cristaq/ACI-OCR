# ACI-OCR
**Install Dependencies**:

To install required Python packages (recommended), run in PowerShell:

```
C:/Users/User/AppData/Local/Microsoft/WindowsApps/python3.13.exe -m pip install -r requirements.txt
```

Or install OpenCV directly:

```
C:/Users/User/AppData/Local/Microsoft/WindowsApps/python3.13.exe -m pip install opencv-python
```

If you still see "No module named 'cv2'", ensure you are using the same Python executable that runs your project (see the command above).
